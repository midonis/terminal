Determine your Terminal size.

## Introduction

### `Terminal.width()`

The `Terminal.width` method returns the current width of the console:

```ts
import { Terminal } from "@midonis/terminal"

Terminal.width()

// 20
```

### `Terminal.height()`

The `Terminal.height` method returns the current height of the console:

```ts
import { Terminal } from "@midonis/terminal"

Terminal.height()

// 80
```
